# Historic Motor Oils / История моторных масел

Этот репозиторий собирает **ключевые моторные масла прошлого века**, их технологические особенности и исторические факты.  
Все записи содержат ссылку на первоисточник для детального изучения.  

This repository collects **key motor oils from the last century**, their technological features, and historical notes.  
Each entry contains a source link for deeper exploration.
